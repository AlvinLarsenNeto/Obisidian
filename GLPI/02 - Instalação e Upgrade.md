## Pré-requisitos
- PHP mínimo 8.2 (para GLPI 11)
- Web server: Apache, Nginx, lighttpd ou IIS
- Banco de dados MySQL/MariaDB compatível
- O GLPI usa o diretório `public/` como base para publicação web

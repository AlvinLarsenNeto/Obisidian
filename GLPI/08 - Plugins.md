# GLPI — Plugins

## Fontes oficiais
- Plugins suportados: https://help.glpi-project.org/doc-plugins
- Configuração de plugins: https://help.glpi-project.org/documentation/modules/configuration/plugins
- Catálogo oficial: https://plugins.glpi-project.org

## Regra de ouro
> **Sempre realizar backup do banco antes de instalar ou atualizar plugins.**

O marketplace de plugins requer chave GLPI Network — inclusive chave gratuita para uso sem assinatura paga.

## Plugins GLPI (gratuitos / open source)

| Plugin | Função |
|---|---|
| Advanced Forms | Formulários avançados de abertura de chamado |
| Carbon | Indicadores de sustentabilidade |
| Centreon | Integração com monitoramento Centreon |
| Credit | Gerenciamento de créditos de suporte |
| Data Injection | Importação de dados em massa via CSV |
| Escalade | Escalonamento de tickets por grupo |
| Fields | Campos customizados em qualquer objeto |
| Formcreator | Catálogo de serviços com formulários |
| Generic Object | Objetos customizados de inventário |
| GLPI Inventory | Inventário (substituto do Fusion Inventory) |
| Inventory Number Generation | Geração automática de números de patrimônio |
| News - Alerts | Alertas e notícias na interface |
| OAuth IMAP | Coleta de e-mails via OAuth (M365/Gmail) |
| Order Management | Gestão de pedidos e compras |
| PDF | Exportação de objetos em PDF |
| SAML | Autenticação SSO SAML |
| SCCM | Integração com Microsoft SCCM/MECM |
| Tag | Etiquetas para qualquer objeto |
| Tree View | Visualização em árvore de ativos |
| Uninstall | Desinstalação/baixa de ativos com workflow |

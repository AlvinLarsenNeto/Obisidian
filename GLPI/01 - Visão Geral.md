# GLPI — Visão Geral

O GLPI centraliza gestão de ativos, service desk, automações e administração. A documentação oficial atual está concentrada no Help Center e dividida entre documentação de usuário, administração/instalação, agent, desenvolvimento, plugins e banco de dados.

## Portais oficiais
- Help Center: https://help.glpi-project.org/
- Documentação principal: https://help.glpi-project.org/documentation
- Documentação de usuário: https://help.glpi-project.org/documentation/readme-1

## Ecossistema oficial
- Documentação de usuário
- Instalação e atualização
- Ferramentas de linha de comando
- GLPI Agent
- Documentação de desenvolvedor
- Plugins suportados
- Esquema de banco de dados

## Observação importante
A própria GLPI informa que a documentação de usuário está sendo reescrita. Manter no Obsidian uma marcação `status: validar` nas páginas mais antigas é recomendado.

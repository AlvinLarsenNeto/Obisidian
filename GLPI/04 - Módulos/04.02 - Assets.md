# GLPI — Assets

## Fonte oficial
- https://help.glpi-project.org/documentation/modules/assets

## Resumo
Módulo para gerenciar hardware, software e demais componentes da infraestrutura — manualmente ou por inventário automático via GLPI Agent. O Fusion Inventory **não é mais suportado** nas versões atuais.

## Tipos de ativos disponíveis
- Computers
- Monitors
- Software
- Network equipments
- Peripherals
- Printers
- Cartridges
- Consumables
- Phones
- Racks
- Enclosures
- PDU
- Passive devices
- Unmanaged assets
- Cables
- SIM
- Global search
